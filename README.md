# Book Discovery System

##  Overview
The **Book Discovery System** is a front-end web application that helps users discover books based on their interests.  
Users can search for books, filter them by genre, view details, and save their favorite books to a wishlist.

This project was built using HTML, CSS, and JavaScript.

---

##  Features
-  Search books by title or author  
-  Filter books by genre  
-  View detailed book information  
-  Add/remove books from wishlist  
-  Wishlist saved using Local Storage  
-  Responsive design for mobile and desktop  

---

##  Technologies Used
- HTML5  
- CSS3  
- JavaScript
